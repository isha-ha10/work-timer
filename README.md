# 🍅 Pomodoro Focus Timer

A simple and clean Pomodoro timer built using Python and Tkinter to help improve focus and productivity.

---

## 🚀 Features

* ⏱️ 25-minute work sessions
* ☕ Short break (5 minutes)
* 🌴 Long break (20 minutes after 4 sessions)
* 🔄 Automatic session switching
* 🎨 Simple and clean UI
* ✔️ Visual progress tracking
* Reset function

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/isha-ha10/work-timer.git
```

2. Navigate to the project folder:

```bash
cd work-timer
```

3. Run the app:

```bash
python main.py
```
